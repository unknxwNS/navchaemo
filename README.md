# Навчаємо — благодійний сайт (UI/UX practice)

Адаптивний маркетинговий сайт благодійної організації **«Навчаємо»** («We Teach»), яка надає якісну освіту дітям з малозабезпечених родин, ВПО, дітям ветеранів та молоді з постраждалих від війни регіонів.

Збудовано на чистому **HTML5 + CSS + vanilla JS** без жодного build-кроку. Усі сторінки відкриваються напряму у браузері через `file://`. Розмітка чиста та семантична — готова до імпорту у Figma через плагін **html.to.design**.

## Структура проєкту

```
navchaiemo-website/
├── index.html            # Головна: герой, анімовані лічильники, проєкти, відгуки, збори
├── about.html            # Про нас: місія, цінності, історія (timeline), команда, партнери
├── projects.html         # Проєкти: сітка карток + робочі фільтри (вік / регіон / вид допомоги)
├── fundraisings.html     # Збори: активні/завершені, donate-модал, «Створити свій збір»
├── documentation.html    # Звітність: фінзвіти, гранти, аудити, реєстраційні документи
├── news.html             # Новини: пошук, фільтр категорій, підписка на розсилку
├── contacts.html         # Контакти: форма з валідацією, карта OSM, QR-коди, реквізити
├── design-system.html    # Бібліотека компонентів (Figma Design System reference)
├── css/
│   └── styles.css        # Спільна дизайн-система (CSS-змінні, теми, компоненти)
├── js/
│   └── main.js           # Спільний JS (без залежностей)
└── README.md
```

## Як відкрити

Просто відкрийте будь-який `.html` файл у браузері (подвійний клік або `Ctrl+O`).
Рекомендований старт — `index.html`. Для перегляду компонентів — `design-system.html`.

> Інтернет потрібен лише для шрифту Inter (Google Fonts), фото з Unsplash та карти OpenStreetMap.
> Без мережі сайт лишається повністю функціональним: спрацьовує локальний fallback-шрифт, а декоративні SVG/патерни вбудовані інлайн.

## Дизайн-система

**Палітра (Ukraine brandbook):**
- Primary (синій): `#0057B8`
- Accent (жовтий): `#FFD700`
- Success (зелений): `#28A745`
- Нейтральні: `#FFFFFF`, `#F8F9FA`, `#E9ECEF`, `#495057`, `#212529`

**Типографіка:** Inter (Google Fonts) з локальним fallback-стеком. Заголовки Bold/Semibold, основний текст 17px.

**Компоненти:** кнопки (primary/accent/secondary/ghost/success), картки зборів, форми донату, іконки (inline SVG), модальні вікна, прогрес-бари, навбар, футер, бейджі, заголовки секцій, glassmorphism-картки.

## Реалізовані сучасні тренди (2025–2026)

- Великі hero-секції з сильним CTA «Допомогти зараз»
- Мікро-анімації та плавні переходи; scroll-reveal через `IntersectionObserver`
- Glassmorphism і м'які тіні на картках
- Інфографіка з анімованими лічильниками (count-up при скролі)
- Темна тема з перемикачем (зберігається у `localStorage`, поважає `prefers-color-scheme`)
- Повна адаптивність (mobile-first), мобільне меню-гамбургер
- Доступність: семантичні лендмарки, `aria`-атрибути, видимий фокус, контраст, alt-тексти, skip-link, `prefers-reduced-motion`

## Примітки

Усі форми, платіжні елементи та QR-коди — демонстраційні (для дизайн-практики); реальні платежі не обробляються. Тексти, назви проєктів, суми, дати та імена — реалістичні, українською мовою.
